# Matthew Moocarme - Personal Website

Portfolio website for Matthew Moocarme, Principal ML Engineer at Paramount Global.

**Live at:** [www.mattmoocar.me](https://www.mattmoocar.me)

## Tech Stack

- Jekyll (GitHub Pages)
- Modern CSS with CSS Grid/Flexbox
- Dark/Light mode support
- Responsive design

## Structure

```
/
├── index.html           # Main single-page site
├── _config.yml          # Jekyll configuration
├── _includes/           # Page section components
├── _layouts/            # Page layouts
├── assets/
│   ├── css/main.css     # Modern stylesheet
│   ├── js/main.js       # Theme toggle & navigation
│   └── img/             # Images
└── Resume/              # PDF resume
```

## Local Development

```bash
bundle install
bundle exec jekyll serve
```

Then open http://localhost:4000
